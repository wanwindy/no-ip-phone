# Round 4 真机问题收敛日报与结论模板

## 1. 目的

Round 4 真机执行结束后，最容易失控的不是“有没有问题”，而是“问题是否进入统一收敛入口、是否有人接、是否影响放行”。本模板用于把真机矩阵、smoke 结果、缺陷分级和放行建议串成一份统一日报或阶段结论。

## 2. 使用规则

1. 每天至少更新一次；如果当天发生换包、复测或新阻塞，建议追加更新。
2. 所有问题都要能追溯到设备记录、证据编号和当前责任人。
3. 已关闭的问题要保留关闭依据，不要直接删除。
4. 不允许把“待确认”“未执行”“等别的同学看看”写成“已收敛”。

## 8. 日报结论模板

建议用下面四句话输出当日结论，方便主线程快速判断：

1. 今日覆盖进度：`最小样本完成 x/y，运营商完成 x/3，smoke 状态如何`。
2. 今日问题状态：`新增问题 x 个，关闭 x 个，开放 P0/P1/P2/P3 各多少`。
3. 当前最大风险：`用一句话写最影响放行的点`。
4. 当前建议：`继续验证 / 限制范围复测 / 暂不建议发布 / 建议发布`。

## 10. 退出条件

只有同时满足以下条件，问题收敛口径才可视为闭环：

1. 最小覆盖计划中的必测项已全部定性。
2. 所有 P0 / P1 都已关闭，或已经明确阻塞发布。
3. 所有开放 P2 / P3 都有责任人和复测时间。
4. smoke 结论与日报结论一致，没有口径冲突。
