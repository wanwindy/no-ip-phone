## 4. 回归策略

### 4.1 必回归触发条件

- 数据模型、迁移、实体字段有变化。
- `/calls/outbound`、callback、event、webhook 任一契约字段有变化。
- 客户端拨号模式、配置开关、状态占位结构有变化。
- 任何缺陷修复触达租户隔离、展示 DID、回拨规则、模式切换四个高风险区。
